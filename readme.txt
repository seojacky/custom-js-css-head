== Description ==

Custom JS&CSS in Head is a lightweight WordPress plugin that allows you to add custom JavaScript and CSS code to the head section of individual posts and pages. Perfect for adding Schema.org markup, custom tracking codes, page-specific styles, or any other custom code that needs to be in the document head.

**Key Features:**

* Add custom JavaScript and CSS code to individual posts and pages
* Clean, intuitive meta box interface in the post editor
* Code is inserted into the `<head>` section with priority 11
* Works with all public post types
* Secure implementation with nonce verification
* Lightweight and performance-optimized
* No database bloat - uses standard WordPress meta fields

**Use Cases:**

* Schema.org structured data markup
* Custom tracking codes (Google Analytics, Facebook Pixel, etc.)
* Page-specific CSS styles
* Custom meta tags
* Third-party integrations
* A/B testing scripts
* Custom fonts loading
